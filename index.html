General Concepts:

Basic concepts of programming
World Wide Web

The World Wide Web is a collection of HTML documents. Those documents in turn are made up of something called HTML - HyperText Markup Language. HTML is the basis of almost every page you encounter. The links between the pages that you see, are called hyperlinks. They are what makes the web, web-like.

Internet

You access the World Wide Web through your computer by using an application called a browser. Computers communicate with other computers or servers using HTTP protocol. A browser's main function is to display the information we typically associate with a website such as text, images, videos and more.

Elements and Tags

HTML documents are made up by HTML elements. Elements are written with a start tag, with an end tag, and with the content in between. The HTML element is everything between the start tag and the end tag.
